face_offset_inner: Optional[float] = None

See outer_offset

face_offset_outer: Optional[float] = 1
Offset is in multiples of tool diameter

  • -1 for closed pockets and inside profiles

  • 0 for open pockets

  • 1 for outside profiles

This offset is applied to any wires and the outer wires of any faces given to the operation.

stepdown: Optional[float] = None

Maximum Z stepdown. A value of None means straight to bottom.

wp: cadquery.cq.Workplane = None

The cadquery Workplane containing faces and/or wires that the profile will operate on.
